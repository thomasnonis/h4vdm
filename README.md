# The project
The scope of this project is to implement the H264 Video Device Matching (H4VDM) paper by Ziyue Xiang, Paolo Bestagini,  Stefano Tubaro and Edward J. Delp.  
The document can be found [here](https://arxiv.org/pdf/2210.11549.pdf).

The project is part of the Multimedia Data Security course held by prof. Giulia Boato at the University of Trento.
