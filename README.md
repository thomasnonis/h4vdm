# The project
The scope of this project is to implement the H264 Video Device Matching (H4VDM) paper by Ziyue Xiang, Paolo Bestagini,  Stefano Tubaro and Edward J. Delp.  
The document can be found [here](https://arxiv.org/pdf/2210.11549.pdf).

The project is part of the Multimedia Data Security course held by prof. Giulia Boato at the University of Trento.

# Approach
At the moment I'm trying to implement an H264 decoder from scratch to extract the required information, but I might (and probably will) move to modifying `openh264` or `ffmpeg` if it becomes too hard/time consuming.
Until that moment, I'll have fun diving into the H264 standard.

Oh and yes, I know it's in Python... it sucks. I'll stick with it for testing and might switch to C/C++ if everything goes well.
